Алгоритм порограммы по подсчету количества реплик по каждой из ролей:
1. Создать пустой объект
2. Пробежать по чекбоксам. Для каждого из них сделать пустой массив в объекте и вставить html в нужный
 html-элемент: "<p>"+роль из чекбокса+"</p>";
3. Пробежать по репликам. При наступлению на каждую из них, определить с какой ролью из чекбоксов 
соотносится данная реплика и увеличить длину соответствующего массива на 1.
4. После цикла: В каждый из абзацсев вставить текст: длина соответствующего массива + "/" + сумма всех
реплик текущего эпизода.
